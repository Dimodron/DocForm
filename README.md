# DocForm — помощник для работы с юридическими документами

Это небольшое десктоп-приложение на **Python + Tkinter**, которое я сделал для своей жены.

Ей было неудобно:

- читать длинные юридические документы;
- параллельно держать открытым шаблон Word;
- вручную копировать куски текста и подставлять их в нужные места отчёта.

Поэтому я собрал одну программу, где слева — документ, а справа — структура будущего отчёта, которая потом автоматически подставляется в корпоративный DOCX-шаблон.

---

## Что умеет программа

### 1. Просмотр исходного файла

Слева в окне можно открыть и просматривать:

- **PDF** (`.pdf`)
- **Word** (`.doc`, `.docx`)
- **Excel** (`.xls`, `.xlsx`)
- **Текстовые файлы** (`.txt`, `.log`, `.md`)

Текст можно **выделять, копировать и вставлять** (Ctrl+C / Ctrl+V, ПКМ).

### 2. Поля для заполнения справа

Справа — “форма” из полей, которые описывают будущий отчёт.

Типы полей:

- `text` — однострочное поле;
- `multiline` — многострочный текст;
- `checkbox` — галочка (в отчёт попадает как «Да», если включена).

Для полей можно:

- **добавлять новые** (внутреннее имя + метка + тип);
- **редактировать** существующие;
- **удалять**;
- **менять порядок** полей (кнопки ↑ / ↓);
- внутреннее имя поля автоматически приводится к **верхнему регистру** (например, `name` → `NAME`).

### 3. Шаблоны полей (профили)

Можно создать несколько разных наборов полей (шаблонов):

- шаблон для одного типа документа;
- другой шаблон для другого кейса.

Возможности:

- переключение между шаблонами через выпадающий список;
- создание нового шаблона на основе текущего;
- удаление шаблонов (кроме того случая, когда остался один единственный);
- у каждого шаблона — свой набор полей и свой DOCX-шаблон.

Все настройки сохраняются в файл **`fields_config.json`** рядом с программой.

### 4. Корпоративный DOCX-шаблон

Отчёт формируется на основе **шаблона Word** (`.docx`), который указывается кнопкой `DOCX шаблон`.

В шаблоне используются **плейсхолдеры вида**:

```text
{{NAME}}
{{ORGANIZATION}}
{{COMMENT}}
```
Так-же все что будет вставлено вместо этих **плейсхолдеров** будет принимать их размер, стиль, и цвет
